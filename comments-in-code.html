What Is an HTML Comment?

A comment starts with <!-- and ends with -->. The browser never renders anything between those two markers; it only shows up if someone views the page's source code: 

<!-- This text is invisible on the page -->

Why Use Comments?

  • To label what a section of markup is for, so the next person reading the file doesn't have to guess.
  • To leave a note or reminder, like a TODO, for whoever edits the page next.
  • To temporarily remove a piece of markup from the page without deleting it outright.

Labeling a Section

Every section on every page of this site is preceded by a comment like <!-- Section: why comments are useful -->, including the one right above this paragraph, so anyone reading the source can find a piece of content without reading the whole file.

A Word of Caution

Comments are only hidden from the rendered page, not from the file itself. Anyone can view a page's source and read every comment in it, so nothing sensitive, like a password or a private note, should ever be written inside one.
