A Meaningful Image

The img tag has no closing tag and no content of its own; everything about it is expressed through attributes. src points to the file, alt describes it for anyone who can't see it, and width and height tell the browser how much space to reserve before the image finishes loading.

A Decorative Image

Not every image carries meaning. When an image is purely decorative, the correct alt value is an empty string, alt="", which tells screen readers to skip right over it instead of announcing a distracting filename.

The Same Image, Resized

Because width and height are just attributes, the same source file can be shown at different sizes on different pages without editing the image itself.
