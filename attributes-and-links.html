Anatomy of an Attribute

An attribute lives inside an opening tag and has three parts: a name, an equals sign, and a quoted value. For example, in <p lang="fr">, the name is lang, the equals sign connects it to the value, and "fr" is the quoted value. Attributes always live inside the opening tag, never the closing tag, and a single element can carry several of them at once.

The lang Attribute

The lang attribute tells the browser and screen readers what language an element's text is written in, which affects pronunciation and translation tools. Here is a French greeting marked up correctly:

Bonjour, comment allez-vous ?

Hover over that sentence (or read its title value) to see a translation, because it also carries a title attribute.

The id and class Attributes

Links: This Section Is Made of Links

A hyperlink is created with the a tag and its href attribute. Here are several different kinds:

  • A fragment link back to the top of this page, using the id="top" on the page's own h1.
  • A relative link to the site's home page, using a plain filename instead of a full web address.
  • An absolute link to MDN's reference on the anchor element, opened in a new tab with target="_blank" and secured with rel="noopener noreferrer".
  • A mailto link that opens the visitor's email client.
  • A tel link that dials a phone number on devices that support it.
